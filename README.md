# UDR-Unidad-diagnostica-rural
## ¿Qué es UDR?
UDR es una empresa ficticia de servicio móvil de resonancia magnética. Su modelo de negocio consiste en acercar la tecnología diagnóstica de alta resolución a poblaciones rurales que carecen de acceso a este tipo de pruebas, evitando desplazamientos innecesarios a hospitales de referencia e integrándose en el circuito de la sanidad pública.

## Páginas
- **Inicio** — Presentación del servicio, ventajas, cómo funciona y llamada a la acción
- **Servicios** — Tipos de estudios, proceso asistencial y tecnología y equipamiento
- **Sobre UDR** — Historia de la empresa, el problema que resuelve y misión y valores
- **Contacto** — Formulario de contacto para solicitar información

## Cómo visualizarla
1. Descarga o clona el repositorio
2. Abre el archivo `index.html` con cualquier navegador web
3. Navega por las páginas desde el menú de navegación

## Tecnologías utilizadas
- HTML5
- CSS3 (diseño responsive con media queries)
